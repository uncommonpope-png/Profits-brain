# 🎨 THE GRAPHICS SOUL

## MISSION
Transform the abstract PLT doctrine into compelling visual assets. The Graphics Soul ensures that every piece of content created by the Content Factory is accompanied by high-quality, brand-consistent imagery.

## CORE CAPABILITIES

### 1. Programmatic SVG Generation
- **Dynamic Icons**: Generate PLT Trinity (Profit, Love, Tax) icons on the fly.
- **Data Visualization**: Create SVG charts for the Revenue Dashboard.
- **Branding Assets**: Generate logos and banners without external dependencies.

### 2. AI Image Orchestration
- **Stability AI Integration**: Generate high-fidelity book covers and social media backgrounds.
- **DALL-E 3 Support**: Create conceptual illustrations for PLT blog posts.
- **Prompt Engineering**: Maintain a database of visual "styles" to ensure Craig's brand remains consistent.

### 3. Visual Branding System
- **Color Palette**:
  - Navy: `#0D1117` (The Base)
  - Pink: `#FF2D78` (The Heart/Love)
  - Cyan: `#00D4FF` (The Tech/Profit)
  - Gold: `#FFB347` (The Reward)
- **Typography**: Clean, monospace, and sans-serif combinations for a "Cyber-Business" aesthetic.

## ARCHITECTURE
The Graphics Soul operates in two modes:
1. **Low-Tax Mode**: Direct SVG generation using template-string injection in Bash or Node.js. (Zero cost, instant).
2. **High-Profit Mode**: API calls to Stability AI for "Immortality-grade" cover art. (Paid, but high conversion).
