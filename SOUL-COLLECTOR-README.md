## How to Use

### Quick Start
```bash
# Start the entire Soul Collector system
./soul-integration.sh start

# View the live bot communication
# Open: http://localhost:8080/bot-communication.html

# View system dashboard  
# Open: http://localhost:8080/soul-dashboard.html
```

### Individual Components
```bash
# Run link validation manually
node link-checker-soul.js start

# Check link checker status
node link-checker-soul.js status

# View health report
node link-checker-soul.js report

# Manage integration system
./soul-integration.sh {start|stop|status}
```
